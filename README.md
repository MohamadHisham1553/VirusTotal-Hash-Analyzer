# VirusTotal-Hash-Analyzer

Python-based threat intelligence utility designed for SOC workflows and malware triage. 

The script automates bulk querying of file hashes against the VirusTotal API (v3), aggregates detection telemetry, and generates a structured Excel report with visual analytics. 
Core Functionality  Accepts file hashes (MD5, SHA1, SHA256) via interactive input or hardcoded list. Queries VirusTotal API with enforced rate limiting (free-tier compliant). 
Extracts detailed attributes: Detection statistics (malicious, suspicious, undetected, harmless) File metadata (name, type, size, hashes) Reputation and community voting Submission and analysis timestamps AV engine detection results (top matches) Normalizes and flattens API response into structured records.  
Detection Logic  Implements a classification model (verdict_style) based on: Malicious count Suspicious count Detection ratio vs total engines Produces four verdict states: Clean Malicious Suspicious Unknown (no data / errors)  Reporting Engine  Generates a multi-sheet Excel workbook using openpyxl:  Hash Report Tabular view of all analyzed hashes Conditional formatting: Color-coded verdicts (green/amber/red/gray) Highlighted malicious detections Full metadata enrichment per hash Auto-filter and structured layout for analyst review Statistics Aggregated distribution of verdict categories Percentage calculations Embedded pie chart for quick risk overview Detection Chart Per-hash breakdown of detection counts Bar chart visualization (malicious, suspicious, undetected)  Operational Characteristics  Handles API errors explicitly (404, 429, network failures). Implements retry-aware delay to avoid rate limiting. Produces analyst-readable output suitable for reporting and escalation. Designed for offline review (Excel artifact).  Use Cases  SOC alert enrichment Malware hash triage Threat intelligence reporting Incident response documentation Bulk reputation analysis of suspicious files  Dependencies  requests for API communication openpyxl for report generation  Output  VirusTotal_Report.xlsx containing enriched intelligence and visual summaries for decision support in security operations.
